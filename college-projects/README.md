## Mangrove Tree Data Analysis (R)

Final project for CIS 351 B (Data Analysis), Spring 2026, done as a team of three. We analyzed lab chemistry data from 126 mangrove samples taken at seven Florida sites, looking at organic matter, carbon, and nitrogen against species and site.

The work covered cleaning the raw data, fixing a unit-conversion mistake in the original researchers' formulas, building visualizations in ggplot2, and running non-parametric tests. A Kruskal-Wallis test showed organic matter differs across sites (p = 3.03e-13), while a Wilcoxon rank-sum test found no significant difference between the two species.

See `mangrove-data-analysis/notes.md` for the full write-up.
